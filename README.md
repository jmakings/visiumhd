# Visium HD End-to-End Spatial Transcriptomics

This repository walks through a single Visium HD spatial transcriptomics
sample from raw 10x output to biological interpretation.

## Workflow
1. QC and filtering
2. Normalization
3. Feature selection
4. Dimensionality reduction
5. Clustering
6. Cell type annotation
7. Differential expression

## Data
Public 10x Genomics Visium HD dataset.
Raw data are not committed to this repository.

## Reproducibility
A Conda environment is provided in `environment.yml`.
